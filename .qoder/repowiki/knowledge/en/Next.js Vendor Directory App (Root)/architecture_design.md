This is the project root for a Next.js app (`next.config.js`, `package.json`). Source lives under `src/` (referenced by the `@/*` → `./src/*` path alias in `tsconfig.json`) and follows the Next.js App Router layout. The build toolchain is: TypeScript via `tsc` (no emit, bundler resolution) + Next.js dev/build/start scripts; styling through Tailwind CSS v4 driven by `@tailwindcss/postcss` in `postcss.config.mjs`; component scaffolding via `shadcn` (`components.json` style `radix-luma`, RSC enabled, aliases `@/components/ui`, `@/lib/utils`, `@/hooks`). Runtime dependencies include `@directus/sdk` for CMS access, `leaflet` + `react-leaflet` for maps, `motion` for animations, `lucide-react` icons, and `next-themes` for dark mode. `next.config.js` hardcodes an unoptimized image remote pattern pointing at `http://206.189.50.71:8055/assets/**`. ESLint uses the flat config format (`eslint.config.mjs`) composed from `eslint-config-next/core-web-vitals` and `eslint-config-next/typescript` with `.next`/`out`/`build` globally ignored.
